Const getPrivateKeyStr

  • getPrivateKeyStr(keypair: XfrKeyPair): Promise<string>
    • Defined in api/keypair/keypair.ts:50

    Returns a private key

    remarks

    Using a given Ledger XfrKeyPair keypair it returns a private key.

    This method is used when user needs to retrieve a private key from the XfrKeyPair keypair

    example
    // We use Findora Ledger to generate a new keypair (in demo purposes)
const ledger = await getLedger();
const keypair = ledger.new_keypair();

// Now, we use `getPrivateKeyStr` to return a provate key of this keypair
const privateStr = await getPrivateKeyStr(keypair);
    throws

    An error returned by Ledger with a prefix added by SDK

    Parameters

    • keypair: XfrKeyPair

      XfrKeyPair

    Returns Promise<string>

    Private key

Const restoreFromPrivateKey

  • restoreFromPrivateKey(privateStr: string, password: string): Promise<WalletKeypar>
    • Defined in api/keypair/keypair.ts:163

    Creates an instance of WalletKeypar using given private key and password.

    remarks

    This method is used to restore a wallet keypair.

    The Keypair contains some essential information, such as:

    • address
    • public key
    • key store

    and so on, and it is used for pretty much any personalized operation that user can do using FindoraSdk

    example
    const password = 'qsjEI%123';
const pkey = 'XXXXXXXXXX';

// Create a wallet info object using given private key and password
const walletInfo = await Keypair.restoreFromPrivateKey(pkey, password);

    Parameters

    • privateStr: string

      Private key

    • password: string

      Password to be used to generate an encrypted KeyStore

    Returns Promise<WalletKeypar>

    An instance of WalletKeypar
